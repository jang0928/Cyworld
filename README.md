### 회원 기능  
---
1. 게시글 작성
 + 게시물 버튼을 통한 게시글 작성 file 업로드 버튼을 통한 사진 추가
2. 프로필 작성
 + 프로필 버튼을 통한 프로필 사진 , 소개글 작성
3. 사진첩 , 게시글 조회
 + 사진이 포함된 게시물과 포함되지 않은 게시글 상세보기
4. 댓글 작성
 + 게시글 및 사진첩에 댓글 달기를 통한 소통
5. 방명록 작성
 + 단순 댓글로만의 소통 기능
6. Total 방문자
 + 본인이 아닌 다른 회원이 방문시 total 횟수 증가
7. 친구 목록
